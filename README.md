# יואב מוד - TNT Prison Block

**Minecraft Bedrock Add-On**

מוד מיוחד שיוצר בלוק TNT מיוחד שתופס ישויות בתוכו!

## תכונות:
- 🎯 בלוק TNT שמתפוצץ וסוגר ישויות
- 💨 חלקיקים דקים אוטפים את הישויות
- 🚶 הישויות יכולות לזוז בחופשיות בתוך הבלוק
- 🔓 בלוק שחרור מיוחד המשחרר את כולם
- 🛡️ הבלוק לא יורד אחרי הפיצוץ

## התקנה:
1. העתק את תיקיית `behavior_pack` ל-`com.mojang/development_behavior_packs`
2. העתק את תיקיית `resource_pack` ל-`com.mojang/development_resource_packs`
3. הפעל עולם חדש והפעל את ה-Add-On

## שימוש:
- בנה בלוק TNT רגיל
- הדליק אותו
- כל הישויות שיפגעו ייתפסו בבלוק
- שים בלוק שחרור לשחרור אותם
